Break:
	breaks out of the while statement
```python

i = 1  
while i < 6:  
  print(i)  
  if i == 3:  
    break  
  i += 1
  
```

Continue
	Basically skips an iteration and continues with the loop
```python

i = 0  
while i < 6:  
  i += 1  
  if i == 3:  
    continue  
  print(i)

```
