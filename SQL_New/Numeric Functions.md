### ROUND
- Rounds the value to its nearest 0
- Syntax
	- ROUND([column_nam],decimal place)
	- Example
		- ROUND(3.16,2) = Rounds it to 3.2
		- ROUND(3.16,1) = Rounds it to 3.0
		- ROUND(3.16,0)  = Rounds it to 3 (Gives whole number)

### ABS
- Gives the absolute value
- Syntax
	- ABS([column_name])
