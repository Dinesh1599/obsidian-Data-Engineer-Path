  1. CONCAT
	- Combines multiple strings
	- Syntax:
		- CONCAT(Firstname,Last Name)
2. UPPER
	- All UPPERCASES
	- Syntax:
		- UPPER(First Name)
3. LOWER
	- All Lower Case
	- Syntax:
		  LOWER(First Name)
4. TRIM
	- Removes Leading and Trailing Spaces(Empty spaces at the beginning and the end)
	- Syntax
		- TRIM(First Name)

### Replace
- Replaces a value with another value
- Can be replacement value can also be blank
- Syntax
	- REPLACE(phnumber,'-','')
